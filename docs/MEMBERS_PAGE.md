# Страница "Члены организации"

## Описание
Новая страница `Members` для управления членами организации с гибкими видами отображения. Страница использует те же данные что и доктора, но предоставляет множество способов просмотра информации.

## Основные функции

### 1. Четыре режима просмотра
- **Grid View (Сетка)** - Карточный вид с основной информацией
- **Table View (Таблица)** - Компактный табличный вид
- **Board by Status (Доска по статусам)** - Канбан-доска с колонками "Активные" и "Неактивные"
- **Board by Region (Доска по регионам)** - Группировка по регионам

### 2. Фильтрация и поиск
- Поиск по имени и специальности
- Фильтр по статусу активности (Все, Активные, Неактивные)
- Динамический счетчик найденных результатов

### 3. Гибкое переключение между видами
- Четыре кнопки с иконками для быстрого переключения
- Интуитивный интерфейс

## Структура страницы

```
Members.tsx
├── MemberCard - компонент карточки члена (Grid view)
├── MemberTableRow - строка таблицы (Table view)
├── MemberBoardByStatus - канбан-доска по статусам
├── MemberBoardByRegion - канбан-доска по регионам
└── Members - главный компонент с логикой фильтрации
```

## Интеграция

### API
Страница использует эндпоинт: `/api/crm/masters/{branchId}`

### Маршрутизация
- Маршрут: `/crm/members`
- Защищенный маршрут через ProtectedLayout

### Навигация
- Добавлена в Sidebar (CRM > Члены)
- Доступна для администраторов и суперадминов

## Поля члена

Используются те же поля что и у Master:
- `id` - уникальный идентификатор
- `name` - имя члена
- `specialty` - специальность
- `description` - описание/примечания
- `isActive` - статус активности
- `startWorkHour` - начало рабочего времени
- `endWorkHour` - конец рабочего времени
- `photoUrl` - URL фото (опционально)
- `region` - регион (опционально, используется для Board by Region)

## Локализация

Добавлены ключи локализации:
- `sidebar.members` - "Члены" (РУ), "Булуштар" (КЫ), "Members" (EN)

## Примеры использования

### Grid View (Сетка)
Показывает карточки членов в 3 колонки на большом экране, удобно для просмотра деталей.

### Table View (Таблица)
Компактный вид с основной информацией в виде строк таблицы, удобно для поиска и сравнения.

### Board by Status
Две колонки: "Активные" и "Неактивные" - удобно для управления статусом членов.

### Board by Region
Колонки по регионам - упрощает работу с членами, распределенными по территориям.
