# Обновление рабочих часов системы

## Изменения

Рабочие часы системы изменены с **8:00-22:00** на **7:00-23:59**.

## Обновленные файлы

### Компоненты календаря

1. **src/pages/Calendar/components/time-schedule.tsx**
   - `generateTimeSlots()`: изменены параметры с `(8, 22)` на `(7, 24)`
   - `getCurrentTimePosition()`: startMinutes изменено с `8 * 60` на `7 * 60`
   - Fallback значения для мастеров: `'09:00'-'20:00'` → `'07:00'-'23:59'`
   - Форма нового сотрудника: дефолтные значения времени обновлены

2. **src/pages/DailyCalendar.tsx**
   - `timeSlots`: startHour и endHour изменены с `(9, 22)` на `(7, 24)`
   - `getCurrentTimePosition()`: границы времени обновлены с `9-22` на `7-24`

### Компоненты управления расписанием

3. **src/components/MasterWorkingDatesManager.tsx**
   - Начальные значения времени: `'09:00'-'18:00'` → `'07:00'-'23:59'`
   - Функция сброса формы обновлена

4. **src/components/MasterWorkingDatesCalendar.tsx**
   - Дефолтные значения формы обновлены
   - Функция создания новой записи обновлена

5. **src/components/MasterScheduleForm.tsx**
   - `EMPTY_SCHEDULE`: время изменено с `'10:00'-'18:00'` на `'07:00'-'23:59'`

### База данных

6. **database_schema.sql**
   - Ограничение времени: `'06:00:00'-'23:00:00'` → `'07:00:00'-'23:59:59'`

## Влияние на функциональность

- ✅ Календарь теперь отображает временные слоты с 7:00 до 23:59
- ✅ Все формы используют новые дефолтные значения времени
- ✅ Валидация времени обновлена для новых границ
- ✅ Линия текущего времени корректно отображается в расширенном диапазоне
- ✅ База данных принимает записи в новом временном диапазоне

## Количество временных слотов

- **Раньше**: 8:00-22:00 = 14 часов × 4 слота = 56 слотов
- **Теперь**: 7:00-23:59 = 17 часов × 4 слота = 68 слотов

Добавлено **12 дополнительных временных слотов** (3 часа × 4 слота по 15 минут).
